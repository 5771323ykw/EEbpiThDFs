# Springboot图书管理系统 java280

## 项目概述

本项目是一款基于Springboot框架的图书管理系统。该系统简单易用，前后端不分离，适合初学者或不需要复杂功能的用户。

## 技术栈

- **后端框架**：Spring Boot
- **数据库**：MySQL
- **开发工具**：IDEA
- **项目管理**：Maven
- **数据库管理**：Navicat

## 功能模块

系统分为以下功能模块：

### 管理员模块

- 读者管理
- 图书管理
- 借阅管理

### 读者模块

- 查看可借阅图书信息

## 系统角色

- 管理员：负责管理读者和图书信息，处理借阅请求。
- 读者：查看图书信息，发起借阅请求。

## 运行环境

- **操作系统**：Windows/Linux/MacOS
- **数据库**：MySQL
- **Java版本**：JDK 1.8及以上

## 部署步骤

1. 环境准备：安装Java、MySQL和Maven。
2. 创建数据库：根据提供的SQL脚本创建数据库。
3. 配置数据库连接：在配置文件中填写正确的数据库信息。
4. 编译项目：使用Maven进行编译。
5. 启动项目：运行Spring Boot应用程序。
